## 📖 Overview | 概述

**orval** is a TypeScript client code generator that transforms [OpenAPI v3](https://www.openapis.org/) or [Swagger v2](https://swagger.io/) specifications (YAML or JSON) into type-safe API clients, hooks, mocks, and more.

**orval** 是一个 TypeScript 客户端代码生成器，可将 [OpenAPI v3](https://www.openapis.org/) 或 [Swagger v2](https://swagger.io/) 规范（YAML 或 JSON）转换为类型安全的 API 客户端、hooks、mock 数据等。

| Feature | 功能 |
|---------|------|
| Type-safe client generation | 类型安全的客户端生成 |

## 📋 Quick Start | 快速开始

### 1. Create `orval.config.ts`

```typescript
import { defineConfig } from 'orval';

export default defineConfig({
  myApi: {
    input: './openapi.yaml',
    output: './src/gen/myApi.ts',
    httpClient: 'axios',
  },
});
```

### 2. Add scripts to `package.json`

```json
{
  "scripts": {
    "generate": "orval --config ./orval.config.ts"
  }
}
```

### 3. Run generation

```bash
pnpm generate
```

## 💡 Usage Demo | 使用示例

See full demo in [demo/demo.ts](demo/demo.ts)

```typescript
// Generated types & client (auto-generated by orval)
// -----------------------------------------
// import { getPetById, createPet, updatePet } from './src/gen/myApi';

// --- Example: React Query Hook ---
async function fetchPet(petId: string) {
  const response = await getPetById({ petId });
  return response.data;
}

// --- Example: with Axios custom instance ---
import axios from 'axios';

const apiClient = axios.create({
  baseURL: 'https://petstore.swagger.io/v2',
});

async function createPetUsingClient(pet: Pet) {
  const response = await createPet(pet, { axios: apiClient });
  return response.data;
}

// --- Example: Zod validation (auto-generated schemas) ---
import { z } from 'zod';

// Schemas are auto-generated from OpenAPI spec
const PetSchema = z.object({
  id: z.number(),
  name: z.string(),
  status: z.enum(['available', 'pending', 'sold']),
});

// Runtime validation
const result = PetSchema.safeParse({ id: 1, name: 'Buddy', status: 'available' });
if (result.success) {
  console.log('Valid pet:', result.data);
}
```

## ⚙️ Configuration | 配置详解

```typescript
import { defineConfig }人啊 from 'orval';

export default defineConfig({
  api: {
    // OpenAPI schema file path
    input: './openapi.yaml',        // or './openapi.json'

    // Output file path
    output: './src/gen/api.ts',

    // HTTP client choice
    httpClient: 'axios',            // 'axios' | 'fetch' | 'hono' | 'angular'

    // Framework
    client: 'react-query',          // 'react-query' | 'vue-query' | 'svelte-query'
                                     // 'solid-query' | 'angular' | 'hono' | 'fetch'

    // Generate mock data
    mock: true,

    // Generate Zod schemas
    schema: true,

    // Clean output directory before generation
    clean: true,

    // Override base URL
    override: {
      mutator: {
        path: './src/custom-client.ts',
        name: 'customClient',
      },
    },
  },
});
```
